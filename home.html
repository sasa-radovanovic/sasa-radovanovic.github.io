Frequent Flyers Flight Diary

Flight Diary Application




Personal Flight Diary to store your flight history and compare worldwide data



For technical data, implementation and download link visit Project Page.

This application is inspired by Flight Diary, Skyscrapercity Forum and my lovely travel companion.

The application deploys two web frontends: public and private one. Public frontend is open for all users, while to access personal page, user must log in.
I will not go into tehnical data here, but just show views from both applications.
